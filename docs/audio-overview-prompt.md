1. オープニング（1分）：ソース末尾の「今日の所感」を基に、今日の世界のAI動向を貫く潮流を要約

3. 高速ダイジェスト：残りの重要度★★☆以上の記事を1件30秒以内で「見出し+一言のなぜ重要か」形式で網羅

- 重要度・関心の格付けはソース内の★表記に従い、独自に再判定しない
